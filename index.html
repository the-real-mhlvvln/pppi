О проекте

Этот проект представляет собой планировщик задач, разработанный с использованием Python, Git и различных инструментов документирования. В рамках проекта были выполнены следующие задачи:

  • Создание базового планировщика задач.
  • Разработка документации в формате DocBook и HTA.
  • Использование веток Git для управления проектом.
